# Prints da interação com o agente de codificação

Coloque aqui os screenshots do uso do Claude Code durante a construção do
RecrutaMe (prompts, iterações, verificação da UI no navegador). Eles são
evidência para o critério **"Uso efetivo do agente de codificação" (4 pts)**.

Sugestões do que capturar:
- O plano por etapas e os portões de aprovação por fase.
- O agente subindo o Streamlit e dirigindo a própria UI (login → Kanban → análise).
- Momentos de refino guiado (function tools, Pydantic, UX).
